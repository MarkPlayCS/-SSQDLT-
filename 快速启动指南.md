### 🎨 现代化界面（推荐）
```bash

python main_modern.py

# 方法二：使用批处理脚本
scripts/启动现代化软件.bat
```

### 📱 原始界面
```bash
# 方法一：直接运行
python main.py

# 方法二：使用批处理脚本

```

## 🎯 快速上手

### 1. 获取数据
- 点击"📊 获取数据"获取最新开奖数据
- 点击"📈 历史数据"获取完整历史数据

### 2. 数据分析
- 点击"🔍 统计分析"进行全面的数据分析

### 3. 智能预测
- 点击"🤖 ML预测"使用机器学习算法预测
- 点击"🧠 DL预测"使用深度学习模型预测

### 4. 保存结果
- 点击"💾 保存结果"导出分析结果
